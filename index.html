Servidores

Seus tipos, características, usos, e exemplos

O que é um servidor?

Um servidor é um sistema de computação que oferece algum dado, programa ou qualquer tipo de serviço para outros computadores, chamados de clientes.

Um servidor pode ser desde um computador doméstico, até uma máquina mais robusta e preparada para um serviço específico. A arquitetura de um servidor geralmente é composta por hardwares como CPUs, memória RAM, discos rígidos ou SSDs, e interfaces de rede para comunicação. Dispositivos que gerenciam o tráfego de dados também são necessários, como roteadores e switches. Firewalls são indispensáveis para a segurança de um servidor, visto que muitas vezes ele é a parte mais importante de um sistema, podendo armazenar quantidades imensas de dados valiosos e sensíveis. Na parte lógica, o servidor precisa de um sistema operacional para rodar, softwares de gerenciamento para ter seu controle, e aplicativos/serviços para funcionalidades específicas.

Tipos e características de um servidor

Backup

Esse tipo de servidor existe para trazer maior segurança, pois seu objetivo é armazenar dados periodicamente, como cópias de segurança, permitindo que dados sejam recuperados.

Data Center

São locais que contém servidores físicos ou virtuais, fornecendo toda a estrutura que esses servidores precisam. Servidores como este geralmente são locais, ficando no local da empresa, o que dá mais controle e segurança.

Dedicado

Este tipo é exclusivo apenas para um cliente ou aplicação específico. Eles possuem desempenho superior, segurança, e dá capacidade de controle e personalização total dele.

Load Balancer

Focado em otimizar servidores, através de distribuir o tráfego de dados entre outros servidores. Assim, ele garante agilidade de resposta e disponibilidade.

Virtual (VPS)

Ele é criado através da virtualização de um servidor físico, permitindo que vários operem de forma independente no mesmo hardware. Isso o permite ter custos menores, tem facilidade de expansão e uma capacidade maior de distribuir cargas de trabalho.

Web

Os servidores web são usados geralmente para a hospedagem de sites e aplicações web em geral, trabalhando com solicitações HTTP/HTTPS de navegadores.

Nuvem

Eles fornecem os mesmos recursos que servidores físicos, porém através da nuvem, eles são mais flexíveis e alta disponibilidade.

Importante – Uptime

Este conceito refere-se ao tempo que um servidor fica no ar, disponível para administradores das aplicações hospedadas e seus externos utilizarem. Isto é, quando todas as funções do servidor estão funcionando corretamente e normalmente, possibilitando suas aplicações continuarem funcionando.

Exemplos

Exemplo de servidor local

Na planta da Bosch em Joinville, que é um escritório administrativo, eles possuem uma sala altamente refrigerada e restrita com um servidor, que rege os andares ocupados pelos colaboradores. A rede, intranet, e todo o controle de dados é feito por esse servidor

Exemplo de servidor remoto

O Amazon EC2 (Elastic Compute Cloud), oferecido pela Amazon Web Services (AWS), é um ótimo exemplo de servidor remoto, que você pode acessar por qualquer lugar do mundo, permitindo que empresas gerenciem seus negócios de forma remote. Sobre seus custos, neste caso são feitos pelo uso de recursos do servidor, sendo pago por segundo ou hora, não há um pagamento fixo de longo prazo, isso torna bem variável o custo por usar um servidor remoto como esse
